## About

Personal portfolio website for Hamid Rafique, BS Cybersecurity student at
Air University, Islamabad (Semester 2, CGPA: 3.09). Built to showcase
projects, certifications, and skills to recruiters and the security community.

---

## Features

- Neural network canvas background (pure Canvas API, no libraries)
- Terminal boot sequence + typewriter role animation
- Glitch effect section headings
- Magnetic 3D card tilt on hover
- Certification filter system (All / DataCamp / Security)
- Copy-to-clipboard contact buttons
- Glassmorphism navbar with active section tracking
- Scroll progress bar
- Fully responsive (mobile, tablet, desktop)
- Zero JavaScript libraries — 100% Vanilla JS

## How to Add New Content

**New Project:** Open `index.html`, find the comment
`ADD NEW PROJECT` and copy the card template above it.

**New Certification:** Find the comment `ADD NEW CERTIFICATION`,
copy a card block, update the name, issuer, date, data-filter
attribute, and the href to the certificate file in `certificates/`.

**New Education Entry:** Find `ADD NEW EDUCATION ENTRY` and
copy a timeline item block.
